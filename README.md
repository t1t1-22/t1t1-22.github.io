# T1T1-22 – Blog and Post Management Guide

This repository hosts content for T1T1-22, including cybersecurity articles, SOC agentics insights, and updates on INCYBER 2026. This guide explains how to create posts, add images, and link external content.

---

## 1. Creating a Post

1. Go to the posts/ folder.
2. Create a Markdown file named:
   YYYY-MM-DD-post-title.md
   Example: 2026-04-02-incyber-2026-lille.md
3. Add front matter at the top of the file:

---
title: "INCYBER 2026 in Lille: Cybersecurity Reimagined"
date: 2026-04-02
author: "T1T1-22"
tags: ["INCYBER", "SOC", "AI", "Agentics"]
---

4. Write your content in Markdown below the front matter.

---

## 2. Adding Images

1. Place images in assets/images/.
2. Reference them in posts using:

![Alt text](../assets/images/image-name.png)

Example:

![INCYBER 2026 conference](../assets/images/incyber-2026-lille.jpg)

---

## 3. Linking External Content

To link to X.com or other sites:

Follow live updates: [Click here](https://x.com/your_post_link)

---

## 4. Formatting Tips

- Headings: #, ##, ###  
- Lists: - for bullets, 1. for numbered  
- Emphasis: *italic*, **bold**  
- Code: `inline` or triple backticks for blocks  
- Links: [text](URL)  
- Images: ![alt text](path-to-image)

---

## 5. Publishing

1. Save changes.  
2. Push to GitHub.  
3. GitHub Pages will automatically update your site.

---

## 6. Best Practices

- Keep titles short and descriptive.  
- Include images for clarity.  
- Use relevant tags.  
- Maintain consistent filenames and dates.
